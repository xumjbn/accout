# 语随记小程序 · 上线配置手册

代码已全部就绪，照本手册操作即可跑通全部功能。标注 🔑 的步骤必须由账号主人操作。

## 第 0 步 🔑 注册小程序，拿 AppID（约 5 分钟，免费）

1. 打开 [mp.weixin.qq.com](https://mp.weixin.qq.com) → 立即注册 → 选「小程序」
2. 邮箱激活 → 主体选「个人」（免费，够用；企业主体才能开支付等能力）
3. 注册完成后：设置 → 基本设置 → 账号信息，复制 **AppID**
4. 填入 `project.config.json` 的 `"appid": ""` 字段

> 个人小程序不能发布「记账理财」类目上架给公众？个人可选「工具」类目，自用+分享给家人（体验版/开发版）完全没问题。

## 第 1 步 导入项目

微信开发者工具（[下载](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)）→ 导入项目 → 目录选 `miniapp/` → AppID 用第 0 步的。

> 此时编译会报「插件未授权」——正常，做完第 2 步就好。着急先看界面的话，把 `app.json` 里 `plugins` 字段临时删掉。

## 第 2 步 🔑 添加同声传译插件（语音识别，约 2 分钟）

1. mp.weixin.qq.com → 设置 → 第三方设置 → 插件管理 → 添加插件
2. 搜索「**同声传译**」（提供方：微信智聆，AppID `wx069ba97219f66d99`）→ 添加
3. 回到开发者工具重新编译。若报版本不存在，把 `app.json` 里 `plugins.WechatSI.version` 改成插件详情页显示的最新版本号

完成后语音记账即可用（支持边说边出字的流式识别）。

## 第 3 步 🔑 开通云开发（家庭共享，约 5 分钟）

1. 开发者工具顶部「云开发」按钮 → 开通 → 选**免费额度**的环境（个人够用）
2. 云开发控制台 → 数据库 → 创建 3 个集合：`families`、`family_members`、`family_tx`
3. 每个集合 → 权限设置 → 自定义安全规则，分别粘贴：

`families`（家庭信息：所有人可查邀请码，仅创建者可改）：
```json
{
  "read": true,
  "write": "doc._openid == auth.openid"
}
```

`family_members`（成员：所有人可读，只能写自己的记录）：
```json
{
  "read": true,
  "write": "doc._openid == auth.openid"
}
```

`family_tx`（共享账单：登录用户可读写——家人要能编辑/删除彼此的账单）：
```json
{
  "read": true,
  "write": "auth.openid != null"
}
```

> ⚠️ 安全边界（MVP 取舍）：以上规则未校验「是否同一家庭成员」，理论上其他使用者构造请求
> 可以读到别人家的账单。自用/家人用没问题；若要公开发布，需把读写收进云函数校验
> `family_members` 成员身份后再操作（客户端不直连数据库）。

## 第 4 步 🔑 配置用户隐私保护指引（真机必需）

小程序用麦克风必须先声明，否则真机上录音授权弹窗不出现：

1. mp.weixin.qq.com → 设置 → 服务内容声明 → 用户隐私保护指引 → 更新
2. 增加处理的信息类型：「**麦克风**」（用途写：语音记账的录音识别）
3. 提交后约几分钟生效

## 第 5 步 验证

- **模拟器**：能记账/预算/资产/统计/导入导出；语音在模拟器可用工具的录音模拟
- **真机**（预览二维码）：语音说「昨天打车花了三十五块五」验证全链路
- **家庭共享双人验证**：A 手机创建家庭 → 复制邀请码给 B → B 加入 → 双方各记一笔 → 互相下拉进明细页看到对方账单

## 常见问题

| 现象 | 处理 |
|---|---|
| 编译报「插件未授权使用」 | 第 2 步没做，或 AppID 不匹配 |
| 语音提示「语音识别不可用」 | app.json 的 plugins 被删了，恢复即可 |
| 家庭页提示需要开通云开发 | 第 3 步没做，或基础库版本过低（工具→详情→本地设置调到 2.2.3+） |
| 真机不弹录音授权 | 第 4 步隐私指引没配置或未生效 |
| 创建家庭报 permission denied | 集合安全规则没贴，或贴错了集合 |
